# RelaySMS-Dashboard

**RelaySMS Dashboard** is an Android Jetpack Compose project that demonstrates a modern, responsive interface for the RelaySMS ecosystem â€” complete with a **Dashboard**, **Language Selector**, and **Team Showcase**.

---

## Features

- **RelaySMS Dashboard**  
  View live stats and controls for syncing messages and managing system data.

-  **Language Selector**  
  Switch seamlessly between **English**, **French**, and **Farsi**, with real-time text updates.

-  **Team Showcase**  
  Beautifully designed team cards, each with unique colors, rounded corners, and role-specific details.
